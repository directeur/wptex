Install

Easy. Follow these steps:

  1. Download WPTeX
  2. Extract the zip file.
  3. Upload the directory 'wptex' to your wordpress directory in your server
  4. Make sure to have wptex directory at the same level as wp-blog-header.php
  5. Make sure that the directory wptex/tmp is writebale. do a chmod 666 on it
  6. Great! Now visit http://yourdomain.com/wptex/ to use WPTeX
